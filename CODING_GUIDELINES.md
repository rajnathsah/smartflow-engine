# SmartFlow Engine - Coding Guidelines\n\nThese guidelines define the coding standards and best practices for SmartFlow Engine development. This document is intended to be referenced by AI code generation tools (Gemini, Claude, etc.) when generating code for this project.\n\n## Table of Contents\n\n1. Code Organization\n- **Single Responsibility Principle**: Each function/class should have one clear purpose\n- **DRY (Don't Repeat Yourself)**: Avoid code duplication\n- **KISS (Keep It Simple, Stupid)**: Write simple, readable code\n- **SOLID Principles**: Follow SOLID design principles when applicable\n\n### 2. Naming Conventions\n- Use **descriptive and meaningful names** for variables, functions, and classes\n- Avoid single-letter variable names except for loop counters\n- Use **English** for all code and comments\n- Use **snake_case** for Python, **camelCase** for JavaScript/React\n\n### 3. Code Comments\n- Write comments for **why**, not **what**\n- Keep comments updated with code changes\n- Use docstrings for all public functions and classes\n- Comment complex logic and non-obvious decisions\n\n### 4. Error Handling\n- Always handle errors gracefully\n- Provide meaningful error messages\n- Log errors with appropriate context\n- Never swallow exceptions silently\n\n### 5. Performance\n- Write efficient code, but prioritize readability\n- Avoid premature optimization\n- Profile code before optimizing\n- Cache expensive operations when appropriate\n\n---\n\n## Python/FastAPI Guidelines\n\n### 1. API Service Layer\n\n```javascript\n// services/api.js\nconst API_BASE_URL = process.env.REACT_APP_API_URL || 'http://localhost:5000';\n\nclass ApiService {\n  static async request(endpoint, options = {}) {\n    const url = `${API_BASE_URL}${endpoint}`;\n    const defaultOptions = {\n      headers: {\n        'Content-Type': 'application/json',\n      },\n    };\n\n    const response = await fetch(url, { ...defaultOptions, ...options });\n\n    if (!response.ok) {\n      throw new Error(`API Error: ${response.statusText}`);\n    }\n\n    return response.json();\n  }\n\n  static get(endpoint) {\n    return this.request(endpoint, { method: 'GET' });\n  }\n\n  static post(endpoint, data) {\n    return this.request(endpoint, {\n      method: 'POST',\n      body: JSON.stringify(data),\n    });\n  }\n}\n\nexport default ApiService;\n```\n\n### 6.
